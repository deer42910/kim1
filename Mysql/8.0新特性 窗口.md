8.0新特性 窗口

```
语法：
  多行函数 over (partition by 列名 order by 列名 desc | asc)
#聚合函数也是窗口函数
#查询所有的商品编号，价格，和类别名以及整体平均价格

SELECT id,NAME,price,category FROM goods;
SELECT AVG(price) FROM goods;#平均价格，只有 只会显示一条
SELECT id,NAME,price,category,AVG(price) over() FROM goods;#多行显示
#查询所有的商品编号，价格，和类别名以及类别平均价格
SELECT id,NAME,price,category,AVG(price) over(partition by category) from goods;
```
